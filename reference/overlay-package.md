# overlay: Render and Overlay ggplot2 and gt objects or Code Snippets with Perspective and Transparency

Renders ggplot2 plots, gt tables, and carbon.js code images as magick
objects, applies optional perspective warp distortions, and composites
them over background images with transparency.
