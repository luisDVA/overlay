# Package index

## All functions

- [`carbon_image()`](https://luisdva.github.io/overlay/reference/carbon_image.md)
  : Create an image of a code snippet with carbon.js via the carbonara
  API
- [`composite_hud()`](https://luisdva.github.io/overlay/reference/composite_hud.md)
  : Composite an overlay onto a background image
- [`hud_overlay()`](https://luisdva.github.io/overlay/reference/hud_overlay.md)
  : Render, warp, and composite a ggplot2, gt, or code snippet over a
  background image
- [`hud_panel()`](https://luisdva.github.io/overlay/reference/hud_panel.md)
  : Wrap a magick image in a dark semi-transparent panel border
- [`render_hud()`](https://luisdva.github.io/overlay/reference/render_hud.md)
  : Render a ggplot2, gt, or carbon code image to a magick image
- [`warp_hud()`](https://luisdva.github.io/overlay/reference/warp_hud.md)
  : Apply a perspective warp to a magick image
